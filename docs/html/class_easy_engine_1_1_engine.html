◆ installEventHandler()

void EasyEngine::Engine::installEventHandler ( std::function< bool(SEvent &)> event_handler)

安装自定义事件处理器

参数
_event_handler用于处理的事件处理器

可使用函数指针、std::function、语法糖等

注解
需要特别注意：
1. 指定的事件处理器必须以 bool 返回。当返回 true 表示持续运行，false 表示结束运行并释放引擎；
2. 原有指定函数指针、 std::function、语法糖等的事件处理器将被替换。
