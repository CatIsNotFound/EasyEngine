详细描述

文本系统

用于加载、渲染字体、支持文本排版功能，实现更为复杂的功能

注解
自 v1.1.0-alpha 版本起，原 FontSystem 重命名为 TextSystem，用于文本排版等功能。
并且文本系统将新增文本排版相关功能。
自从
v1.1.0-alpha
