详细描述

通用序列化和反序列化框架

该框架支持：

  1. 基本数据类型（int, float, bool等）
  2. std::string类型
  3. 嵌套结构体和类
  4. 标准容器类型（std::vector, std::list, std::map等）
  5. 智能指针（std::shared_ptr, std::unique_ptr）

核心思想是使用模板特化和SFINAE技术为不同类型提供专门的处理。
