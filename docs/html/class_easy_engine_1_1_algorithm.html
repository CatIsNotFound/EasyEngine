◆ compareEllipse()

int8_t Algorithm::compareEllipse ( const Graphics::Ellipse & ellipse1,
const Graphics::Ellipse & ellipse2 )
static

判断椭圆与椭圆之间的位置关系

参数
ellipse1指定第一个椭圆
ellipse2指定第二个椭圆
返回
返回一个值以表示不同的位置关系
返回值
-1表示两个椭圆完全分离
0表示两个椭圆属于相切或相交
1表示 ellipse2 的中心点包含 ellipse1
2表示 ellipse1 的中心点包含 ellipse2
注解
此函数对于近似的圆形效果更好。
对于椭圆，会随着宽高之差的增加而增加判断误差！

返回
