◆ draw()

void Components::Layer::draw ( bool viewport_mode = false,
bool clip_mode = false )

绘制图层

参数
viewport_mode是否使用视图（需调用 setViewport()）
clip_mode是否使用裁剪视图（需调用 setClipViewport()）
注解
执行此函数前，若 viewport_mode 或 clip_mode 为 true，
还需调用 setViewportPainter() 才能生效！

执行后，这将会按照图层的渲染顺序依次绘制！z_order 越大，渲染越靠后，显示最靠前！
