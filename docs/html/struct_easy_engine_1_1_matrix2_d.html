◆ multiply()

template<typename T>
void EasyEngine::Matrix2D< T >::multiply ( const Matrix2D< T > & other)

全局乘法

参数
other指定矩阵，其指定的行数必需与现有的列数相等

将矩阵里的所有值进行乘法操作

注解
当前仅支持整数、浮点数运算，不支持其它数据类型的运算！
警告
两个矩阵必需分别为 m * n, n * p 的大小才可用！否则将异常报错！

翻转矩阵

参数
reverse_row行与行之间逆序（垂直翻转）
reverse_col列与列之间逆序（水平翻转）
注解
当两个参数都为 true 时，矩阵将完全逆序

当选择任何一个方向的逆序时，都会发生交换。
